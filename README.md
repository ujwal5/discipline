# 🏆 UJWAL — Discipline OS

A personal discipline tracker web app. Track daily habits, log spending, and visualize your weekly progress.

## Features
- ✅ One-click habit tracking (Sleep, Wake, Exercise, No PMO, Healthy Eating)
- 💸 Daily outside spend logging (₹)
- 📊 Charts: daily scores, habit breakdown, spend trends
- 📚 Full week history with week-by-week comparison
- 📱 Mobile-first dark UI — works on phone via GitHub Pages
- 💾 All data saved locally in browser (localStorage)


> ⚠️ Data is stored in your **browser's localStorage** — it stays on each device separately. For cross-device sync, open on each device and use it there.

## Usage
- **TODAY tab** — tap a habit card to mark it done (turns green ✓)
- **Spend field** — enter ₹ amount if you spent outside; leave 0 if not
- **Notes** — write a daily reflection
- **HISTORY tab** — see all past weeks + comparison charts
- **STATS tab** — overall lifetime stats and habit breakdown
